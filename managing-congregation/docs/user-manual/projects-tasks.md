# Projects & Tasks

The System allows you to manage community projects, renovations, or events, and assign tasks to members to ensure accountability.

![Projects Dashboard - Screenshot](../images/projects-dashboard.png)

Projects are larger initiatives (e.g., "Renovation of the Chapel", "Summer Camp 2024").

### Creating a Project

1.  Go to **Projects**.
2.  Click **Create Project**.
3.  **Details**:
    -   **Name**: Clear title.
    -   **Budget**: Estimated cost (optional).
    -   **Timeline**: Start and End dates.
4.  Click **create**.

![Create Project Modal - Screenshot](../images/project-create.png)

### AI Assistance (Beta)

Stuck on planning? Use the built-in AI assistant.

1.  Click the **AI Create** button.
2.  **Describe your goal**: e.g., _"Plan a 3-day spiritual retreat for 50 youth"_.
3.  The AI will generate a structured project plan with suggested phases and tasks.
4.  Review and Click **Import** to add it to your system.

![AI Project Assistant - Screenshot](../images/project-ai-assistant.png)

Break down projects into actionable steps.

### Task Board (Kanban)

View tasks in columns: **Pending**, **In Progress**, **Completed**.

-   **Drag and Drop** cards to update their status.

![Kanban Board - Screenshot](../images/project-kanban.png)

### Assigning Tasks

1.  Click inside a Project.
2.  Click **Add Task**.
3.  Enter the description and **Assignee** (select a member).
4.  Set a **Due Date**.

### My Tasks

Each user has a personal view.

-   Click **My Tasks** in the menu to see everything assigned specifically to _you_ across all projects.

![My Tasks View - Screenshot](../images/my-tasks-list.png)
