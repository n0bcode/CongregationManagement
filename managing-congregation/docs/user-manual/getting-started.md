# Getting Started

Welcome to the Congregation Management System. This guide will help you navigate the basic features of the application.

## ðŸ  Dashboard Overview

Upon logging in, you are greeted with the main Dashboard. This central hub provides an immediate snapshot of the community's status.

![Dashboard Overview - Screenshot](../images/dashboard-overview.png)
_Figure 1: Main Dashboard view showing quick stats and upcoming events._

The dashboard is divided into three key areas:

1.  **Quick Stats Card**: located at the top, showing total counts:
    -   **Members**: Total number of active members.
    -   **Communities**: Number of active houses.
    -   **Active Users**: Currently logged-in system users.
2.  **Upcoming Birthdays**: A list of members celebrating birthdays in the next 30 days. Clicking on a name takes you to the Celebration page.
3.  **Recent Activities**: A real-time log of system actions (e.g., "Sister Maria added a new health record").

## ðŸ§­ Navigation

The sidebar (or top bar on mobile) provides access to all major modules.

![Sidebar Navigation Menu - Screenshot](../images/sidebar-menu.png)
_Figure 2: The main navigation menu._

-   **Dashboard**: Return to this main view.
-   **Members**: Manage member profiles, formation stages, and search.
-   **Communities**: Manage houses, contact info, and assignments.
-   **Projects**: Track community projects, tasks, and AI plans.
-   **Financials**: Manage expenses, budgets, and view financial reports.
-   **Reports**: Access demographic data and annual community summaries.
-   **Celebrations**: Birthday card generation and management.
-   **Settings** (Admin only): System configuration and backups.

## ðŸ” Login & Safety

### Logging In

1.  Enter your **Email Address**.
2.  Enter your **Password**.
3.  Click **Login**.

![Login Screen - Screenshot](../images/login-screen.png)

### Managing Your Profile

To update your password or personal details:

1.  Click your **Name** in the top right corner.
2.  Select **Profile**.
3.  You can update your **Password** or enable **Two-Factor Authentication** (if configured).

![User Profile Settings - Screenshot](../images/profile-settings.png)

> [!NOTE]
> For security, your session will automatically time out after a period of inactivity. Please remember to log out when using a shared computer.
