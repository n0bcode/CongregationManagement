# Reports

Access these from the main **Reports** menu.

### Demographic Report

**Purpose**: Analyze the composition of the congregation.

-   **Age Distribution**: Visual charts of age groups (e.g., 20-30, 30-40).
-   **Vows Status**: Percentage of Temporary vs. Perpetual Vows.
-   **Nationality**: Breakdown by country of origin.
-   **Download**: Export full data as CSV for Excel or PDF for printing.

![Demographic Report Charts - Screenshot](../images/report-demographic.png)

### Community Annual Report

**Purpose**: A yearly summary for a specific community, often required for provincial archives.

1.  Select the **Community**.
2.  Select the **Year** (e.g., 2024).
3.  The report lists:
    -   Residents at the start of the year.
    -   Arrivals and Departures.
    -   Major events recorded.

![Community Annual Report - Screenshot](../images/report-community-annual.png)

The **Report Builder** allows you to create custom queries to answer specific questions.

1.  Go to **Reports > Builder**.
2.  **Add Filters**:
    -   _Example_: "Status is Active" AND "Age > 60" AND "Location is Rome".
3.  **Run Query**: The results table appears below.
4.  **Export**: Save the table as a CSV file.
