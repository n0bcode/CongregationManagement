## ðŸ§¾ Expenses

Track every expenditure to ensure transparency.

### Recording an Expense

1.  Navigate to **Financials > Expenses**.
2.  Click **Add Expense**.
3.  **Fill Details**:
    -   **Amount**: Value in local currency.
    -   **Date**: Date on the receipt.
    -   **Category**: Choose from "Food", "Medical", "Travel", "Maintenance", etc.
    -   **Description**: Brief note (e.g., "Weekly grocery run").
    -   **Project** (Optional): Link this expense to a specific Project.
4.  Attach a photo of the receipt (Optional).
5.  Click **Save**.

## ðŸ”’ Monthly Closing & Reports

To ensure data integrity, financial periods can be "locked".

### Locking a Period

_Role Required: Financial Administrator_

1.  Go to **Financials > Settings**.
2.  Select a month to **Lock**.
3.  Once locked, expenses in that month cannot be edited or deleted.

### Monthly Report

Generate a PDF report for submission to the higher council.

1.  Go to **Financials > Reports**.
2.  Select **Monthly Summary**.
3.  Select the Month/Year.
4.  Click **Download PDF**.
